# 2017.2 Revendedora

## Sobre o Projeto

## Principais Features

O projeto Revedora tem como principais features:

* Cadastro de Fornecedores 
* Cadastro de Clientes
* Gerenciamento dos clientes 
* Criação e gerenciamento de itens cosméticos
* Realizar criação de kits
